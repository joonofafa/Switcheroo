
## 버전 1.1.0.0에 추가할 사항
    1) "Executer" 입력창에서 ":"을 입력할 경우 현재 윈도우에서 접근 가능한 "드라이브"들이 하단에 리스트에 표시되며, 사용자가 선택 가능해야 함 
       "드라이브"를 선택하면 "Execute" 입력창에 드라이브 문자가 표시되며, 드라이브의 루트에 있는 디렉토리와 파일들이 표시되어야 함.
    2) 사용자가 입력창에 "C:\", "D:\"등 드라이브 문자를 직접 입력해도 1)과 동일하게 동작해야 함.
    3) 디렉토리와 파일들이 표시될 때, 디렉토리와 파일 목록의 아이콘은 Explorer에 표시되는 아이콘 및 파일명과 동일해야 함.
    4) 디렉토리와 파일들이 표시되고, 아래/위 화살표로 선택하면 사용자 입력창에 해당 경로가 텍스트로 표시 됨.
    5) 해당 경로는 Ctrl+A, Ctrl+C로 복사 가능하며, Ctrl+V로 경로가 붙여졌을 경우도 해당 경로의 디렉토리와 파일들이 표시되어야 함.
    6) 하단 리스트에 표시되는 파일을 아래/위 화살표로 선택 가능한 상황에서 파일을 선택하여 [입력(Enter)]키를 누르는 경우, 해당 파일은 Shell로 "열기"처리가 되어야 함
    7) 하단 리스트에 표시되는 디렉토리를 아래/위 화살표로 선택 가능한 상황에서 파일을 선택하여 [탭(Tab)]키를 누르는 경우, 
       해당 디렉토리로 이동하며, 해당 경로는 사용자 텍스트에 표시되며 이후 동작은 동일하게 동작 함.
    8) 리눅스의 Bash 처럼, 힌트글자를 입력 후, [탭(Tab)]키를 누르는 경우에는 해당 글자에 해당하는 디렉토리 또는 파일 목록들로 필터가되며, 
       목록이 1개인 경우, 해당 경로로 경로가 완성되어야 한다.
    9) Switcheroo 실행 파일이 있는 위치의 func_list.json 파일을 읽어들여서 Function키에 따라 특수기능에 따라 동작해야한다.
       func_list.json은 다음과 같이 구성될 수 있다.
        "
        [
            {
                "key": "F5",
                "process_list": "",
                "execute_list": 
                    {
                        "func": "PATH_MARCO",
                        "value": "E:\Workspace\GitHub\Switcheroo\Switcheroo\bin\Release"
                    }
            },
            {
                "key": "F6",
                "process_list": "",
                "execute_list": 
                    {
                        "func": "PATH_MARCO",
                        "value": "C:\Users\JHSOFT\Downloads"
                    }
            }
        ]
        "
        - "func"에 해당하는 기능동작은 향후 추가될 특수기능들에 따라 추가로 정의될 수 있다. (Enum등으로 선언 필요)
        - 예시에 명시된 PATH_MARCO는 사용자 입력창에 "value"에 해당하는 경로를 즉시 붙여넣기하고, 해당 경로의 디렉토리/파일여부에 따라 내용을 리스트로 보여줘야 한다. (기존 동작과 동일)
